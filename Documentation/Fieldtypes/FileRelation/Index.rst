Additional Configuration Options
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

* **allowed**
Comma-separated list of allowed file types

* **disallowed**
Comma-separated list of disallowed file types

* **showUpload**
Show the upload field for Drag & Drop the files directly onto the form
